O **Sentinela** é um sistema de detecção e resposta a intrusão (IDS) em tempo real, baseado em arquitetura event-driven com Kafka, Python e PostgreSQL.

---

## 🚀 Diferenciais Tecnológicos

- Kafka em modo KRaft (sem Zookeeper)
- Rule Engine stateful com análise por IP
- Classificação dinâmica de risco
- Persistência desacoplada com PostgreSQL
- Dashboard em tempo real estilo SOC
